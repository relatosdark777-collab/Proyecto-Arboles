# Sistema de Inventario "Tree-Stock"

## Objetivo
Gestionar un inventario de productos mediante una estructura de Árbol Binario de Búsqueda (ABB) implementada en Java.

## Comprensión Teórica
- **Árbol Binario de Búsqueda (ABB):** Estructura jerárquica donde los nodos menores van a la izquierda y los mayores a la derecha.
- **Recursividad:** Técnica utilizada para insertar y buscar productos recorriendo las ramas del árbol de forma eficiente.
- **Punteros:** Referencias manuales (`izquierdo` y `derecho`) que conectan los objetos Producto en la memoria.

## Instrucciones de Ejecución
1. Abrir terminal en la carpeta del proyecto.
2. Compilar: `javac Main.java`
3. Ejecutar: `java Main`
